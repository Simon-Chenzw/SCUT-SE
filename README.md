## 描述
全部代码由本人写于2019学年下学期，仅供参考  
**不保证代码100%正确**，只保证可以通过编译，发现问题欢迎提issue  
由于本人的编程习惯，不太喜欢多文件，除去部分代码量较大的题目，均为一个文件。推荐自己分离声明和实现  
多文件的题目，均提供了Makefile供一键编译。
